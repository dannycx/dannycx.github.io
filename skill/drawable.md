# XML 图片选择器、shape图形

## selector选择器
#### 按钮选择器
```java
<?xml version="1.0" encoding="utf-8"?>
<selector xmlns:android="http://schemas.android.com/apk/res/android">
    <item android:state_pressed="true" android:drawable="@drawable/pressed"/>
    <item android:drawable="@drawable/normal"/>
</selector>
```
